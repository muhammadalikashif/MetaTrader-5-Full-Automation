# Local MT5 Launch Configuration

This folder stores MT5 launch configuration files generated at runtime.

## Files

- `launch_autotrade1.ini` — Generated by `generate_launch_config.py`.
  Contains account credentials (login, password, server) for the MT5
  startup config.

## WARNING

**Do not commit `launch_autotrade1.ini`** if it contains account credentials.

Credentials are local-machine specific. The `.gitignore` should include
`config/` or `launch_autotrade1.ini` to prevent accidental commits.

## How to regenerate

If this folder is empty, run:

```
python run_monitor.py launch-autotrade1
```

This generates the INI file automatically during the launch pipeline.
